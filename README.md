# FreeSearch

Self-hosted Search Engine Android App - No content filtering.

## Features
- 🔍 Multi-engine search (DuckDuckGo, Brave, Startpage, Qwant, Custom)
- 🚫 No content filtering / SafeSearch disabled
- 🌙 Dark mode support
- 📱 Android 7+ (API 24+)
- 🔐 APK Signed v2+v3
- 📜 Search history

## Build
APK is automatically built via GitHub Actions. See Releases for download.

## Tech Stack
- Kotlin
- Android SDK 34
- Material Design 3
- WebView
